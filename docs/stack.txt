Para a realização deste desafio, pretendo utilizar as seguintes tecnologias e práticas:

- Typescript, Node.js (com Express), React e PostgreSQL.
- Utilizarei um ORM como o Sequelize para facilitar a interação com o banco de dados PostgreSQL.
- Adotarei as melhores práticas de engenharia de software, como:
-- Reutilização de Código: Criarei componentes e funções reutilizáveis para otimizar o desenvolvimento e a manutenção do código.
-- Testes Unitários: Implementarei testes unitários abrangentes para garantir a qualidade e a robustez do código.
-- Documentação: Elaborarei uma documentação clara e concisa, tanto do código quanto da aplicação, facilitando a compreensão e manutenção por outros desenvolvedores.
- Diferenciais: Se o tempo permitir, explorarei as seguintes possibilidades:
-- Manifestos Kubernetes: Criarei manifestos para a orquestração da aplicação em um ambiente Kubernetes, visando escalabilidade e alta disponibilidade.
-- CI/CD: Implementarei uma pipeline de CI/CD para automatizar o processo de build, testes e deploy da aplicação.
